# Gemini-Image-und-Prompt v1.0.0

## Problem
Turning a URL into a high-quality social asset typically requires multiple tools and manual prompt engineering. This release packages extraction, prompt generation, and image rendering into one reproducible workflow.

## Core Features
- URL scraping with SSRF validation and provider fallback
- Prompt generation with Gemini streaming responses
- Imagen4 rendering with credit-aware gating
- Stripe-backed credit packs and webhook reconciliation
- Built-in abuse controls (rate limits, daily caps, regeneration limits)
